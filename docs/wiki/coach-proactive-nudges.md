# Coach Proactive Nudges

## Proactive Coaching Strategy

Proactive coaching should dominate interactions, with roughly 80% of interactions coach-initiated rather than reactive. Use `schedule_nudge` aggressively to drive key daily touchpoints: morning weigh-in replies, pre-meal nudges, and evening reviews. <!-- [^cd8bc-5] -->

## Trigger-Based Nudges

A weight-save event triggers the coach agent to generate a proactive morning response. An evening review is scheduled as a nightly local notification. <!-- [^cd8bc-6] -->
