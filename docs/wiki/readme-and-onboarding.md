# README and Onboarding

## README Strategy

The README focuses on getting people excited about the app rather than explaining technicals, and uses screenshots to explain features. The deficit-from-scale feature has a dedicated README section because it is unique among apps and serves as a motivation driver; this section frames the deficit as coming from the scale rather than from logging, using phrasing like 'reading the evidence your body left behind'. The README text is skimmable, using bolded lead lines per section, shorter paragraphs, bulleted lists for key sections, and bolded numbers, without being overdone. The README includes a link to the TestFlight beta at https://testflight.apple.com/join/muNV9QP5.
