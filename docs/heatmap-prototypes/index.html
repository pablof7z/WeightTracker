1 · Segmented Thermometer + Trajectory

Color = position of your 7-day moving-average weight relative to the linear target trajectory for that day. Cividis ramp from dark (ahead) to pale (behind).

2 · Segmented Thermometer + Binary + Glyph + Sparkline

Cell = weighed-in (filled) / missed (outlined) / future (faded). A small ▼▬▲ glyph encodes direction vs. yesterday. Magnitude lives in the sparkline of the smoothed 7-day MA.

3 · 7×N Grid + Trajectory

GitHub-style 7-row contribution grid, one column per week. Same cividis trajectory rule as Prototype 1. Hairline now-line splits past from future.

4 · 7×N Grid + Binary + Direction-Saturation + Sparkline

Same grid as Prototype 3. Cells are binary (filled / missed / future); direction vs. yesterday encoded as fill saturation (lighter = up, darker = down) since glyphs don't fit at 14px. Magnitude lives in the sparkline.
